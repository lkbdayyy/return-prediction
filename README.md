# 📦 E-commerce Returns Prediction Project

Dự án phân tích hành vi trả hàng trong thương mại điện tử.

## 📁 Cấu trúc dự án (Project Structure)

Dự án được tổ chức theo quy trình chuẩn Data Science:

* **notebooks/** : Chứa toàn bộ mã nguồn phân tích từng bước.
* **outputs/** : Kho lưu trữ 10 biểu đồ trực quan hóa kết quả.
* **src/** : Các module xử lý dữ liệu dùng chung.
* **data/raw/** : Dữ liệu thô đầu vào.

## 📊 Phân tích chi tiết trong Notebooks

Mời giảng viên và người xem truy cập thư mục `notebooks/` để xem chi tiết:

1. `01_eda.ipynb` : Phân tích 10 biểu đồ hành vi.
2. `02_preprocess_feature.ipynb` : Kỹ thuật tiền xử lý dữ liệu.
3. `06_evaluation_report.ipynb` : Đánh giá hiệu quả mô hình dự báo.
